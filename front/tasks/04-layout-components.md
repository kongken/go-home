# Task 4: 布局组件

## 目标
实现 uchome 风格的主布局

## 组件

### 1. Header (顶部导航)
- Logo/站点名称
- 搜索框
- 导航链接: 首页、博客、群组、消息
- 用户菜单: 头像下拉 (个人主页、设置、退出)

### 2. Sidebar (侧边栏)
- 用户信息卡片 (头像、昵称、简介)
- 快捷导航: 我的博客、我的好友、我的相册
- 统计信息: 博客数、好友数

### 3. MainLayout (主布局)
- Header 固定顶部
- 内容区: 左侧 Sidebar + 中间内容 + 右侧可选 Widget
- 响应式: 移动端隐藏 Sidebar

## 需要组件
- shadcn: Avatar, DropdownMenu, Input, Button, Badge, Separator

## 输出
- Header 组件
- Sidebar 组件
- MainLayout 组件
