# Task 1: 初始化前端项目

## 目标
使用 React + Vite + TypeScript + shadcn/ui 初始化前端项目

## 步骤
1. 使用 vite 创建 React + TypeScript 项目
2. 安装和配置 shadcn/ui
3. 安装基础依赖 (react-router-dom, axios, zustand 等)
4. 配置项目结构

## 命令
```bash
cd /home/node/workspace/go-home/front
npm create vite@latest . -- --template react-ts
# 安装 shadcn
npx shadcn@latest init
# 安装基础依赖
npm install react-router-dom axios zustand @tanstack/react-query
```

## 输出
- 完整的 React + Vite + TypeScript 项目结构
- shadcn/ui 配置完成
- 基础依赖安装完成
